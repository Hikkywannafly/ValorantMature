## Important Notice

This tool modifies local game package files during runtime. It is not affiliated with Riot Games, VALORANT, or VNG. Use it at your own risk.

## Main Risks

| Risk | Explanation | Safer Practice |
| --- | --- | --- |
| Game update or repair | Updates may expect original files to be present. | Restore before updating or repairing. |
| Missing backup | Restore depends on the local backup folder. | Do not delete or move `backup`. |
| Forced shutdown | Closing the app or Windows abruptly can interrupt restore. | Close the game normally, then exit the app. |
| Wrong folder | Selecting the wrong folder can make the tool ineffective. | Select the actual VALORANT `Paks` folder. |
| Account/game policy | Any third-party modification may carry policy risk. | Review the risk and decide for yourself. |

## Restore Checklist

Restore original files before:

- running game repair,
- updating VALORANT,
- moving the tool,
- deleting the backup folder,
- uninstalling the tool.
