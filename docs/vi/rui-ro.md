# Rủi Ro

[Về README](../../README.md) · [Discord hỗ trợ](https://discord.gg/DXX4x5TQRq)

## Lưu Ý Quan Trọng

Tool này thao tác file package cục bộ khi game chạy. Dự án không liên kết với Riot Games, VALORANT hoặc VNG. Bạn tự chịu trách nhiệm khi sử dụng.

## Rủi Ro Chính

| Rủi ro | Giải thích | Cách an toàn hơn |
| --- | --- | --- |
| Update hoặc repair game | Game có thể cần file gốc trước khi update/repair. | Khôi phục trước khi update hoặc repair. |
| Mất backup | Khôi phục phụ thuộc vào thư mục `backup`. | Không xoá hoặc di chuyển `backup`. |
| Tắt máy/app đột ngột | Restore có thể bị gián đoạn. | Thoát game bình thường rồi mới tắt app. |
| Chọn sai thư mục | Tool không tác dụng hoặc không tìm thấy file. | Chọn đúng thư mục `Paks` của VALORANT. |
| Chính sách game/tài khoản | Mọi chỉnh sửa bên thứ ba đều có rủi ro chính sách. | Tự cân nhắc trước khi dùng. |

## Checklist Khôi Phục

Nên khôi phục file gốc trước khi:

- repair game,
- update VALORANT,
- di chuyển tool,
- xoá thư mục backup,
- gỡ tool khỏi máy.

## Không Cam Kết An Toàn Tuyệt Đối

Source public giúp minh bạch cơ chế, nhưng không phải cam kết an toàn tuyệt đối. Bạn là người quyết định và chịu trách nhiệm khi sử dụng.
